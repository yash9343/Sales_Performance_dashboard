# Sales Performance Dashboard

## Project Overview
This project analyzes sales data to find business insights
using Python, SQL, Excel and Power BI.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- MySQL Workbench
- Power BI
- Excel

## Key Insights
- West region has highest sales
- Technology category is most profitable
- Q4 shows highest sales growth

## Files
- sales_data.xlsx - Raw dataset
- sales_data_cleaning.py - Data cleaning script
- sales_performance_analysis.sql - SQL queries
- sales_performance_dashboard.pbix - Power BI dashboard
